# How to Run Examples

## Requirements

Make sure you have installed all the requirements for Jaiger.

## Simple App

Simply run `simple_app.py`

`python examples/simple_app/simple_app.py`

## Web UI App

1. Install web UI requirements.

`pip install -r examples/web_ui_app/requirements.txt`

2. Run `web_ui_app.py`

`python examples/web_ui_app/web_ui_app.py`
